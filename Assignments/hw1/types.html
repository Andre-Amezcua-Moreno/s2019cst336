Types:

1. Boot Sector Virus

This type of virus takes control when you start or boot your computer.

Spread when you plug an infected USB drive into your computer.

2. Web Scripting Virus

This type of virus resides in the code of web browsers and web pages.

Spreads when you access a web page that contains the virus, infectint your computer.

3. Browser Hijacker

This type of virus “hijacks” certain web browser functions.

It automatically directs you to an unintended website.
